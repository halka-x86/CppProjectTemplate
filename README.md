# CppProjectTemplate
c++ project template

## Description
個人用の c++ 開発用テンプレート．
vscode での開発を前提として.vscodeを作成している．
(Ctrl+Shift+B で make&実行 を行うタスクを設定)

    ディレクトリ構成

    ├─ makefile
    ├─ bin       # 実行ファイルが生成されるディレクトリ
    ├─ include   # インクルードディレクトリ
    ├─ obj       # 中間ファイル生成先ディレクトリ
    ├─ src       # ソースファイルディレクトリ
    └─ README
